# 🧠 AI 음악 추천 스크립트 (`script.js`)

본 프로젝트의 핵심 로직을 담당하는 자바스크립트 파일입니다. 사용자의 입력을 받아 Google Gemini API와 통신하고, 그 결과를 웹 화면에 실시간으로 반영(렌더링)하는 기능을 수행합니다.

---

## ⚙️ 주요 기능 및 로직 흐름 (Logic Flow)

1. **DOM 요소 및 데이터 검증**
   * 입력창(`userInput`), 결과창(`resultBox`), 로딩 스피너(`loader`) 요소를 가져옵니다.
   * 사용자가 아무것도 입력하지 않고 버튼을 누르면 경고창(`alert`)을 띄워 흐름을 차단합니다.

2. **화면 상태 제어 (UI State)**
   * AI가 답변을 생성하는 동안 대기 시간을 시각적으로 보여주기 위해 로딩 스피너를 활성화(`display: block`)하고, 이전 결과창은 숨깁니다.

3. **Google Gemini API 통신 (비동기 Fetch)**
   * 최신 `gemini-2.5-flash` 모델 엔드포인트로 `POST` 요청을 보냅니다.
   * 프롬프트 엔지니어링을 통해 AI가 일반 텍스트가 아닌, 미리 정의된 **유튜브 검색 링크가 포함된 HTML 구조**로 답변하도록 강제합니다.

4. **데이터 정제 및 화면 렌더링 (Data Parsing)**
   * AI 응답에 포함될 수 있는 마크다운 기호(```html ... 
```)를 정규식(`replace`)으로 깨끗하게 제거합니다.
   * 정제된 HTML 소스코드를 `innerHTML`을 통해 결과창 박스에 그대로 주입하여 동적 화면을 완성합니다.

---

## 🛠️ 함수 및 변수 설명 (API Reference)

* **`GEMINI_API_KEY`** (Constant)
  * Google AI Studio에서 발급받은 인증 키 문자열입니다.
* **`getMusicRecommendation()`** (Async Function)
  * 버튼 클릭 이벤트와 연결된 메인 비동기 함수입니다. API 호출부터 예외 처리, 화면 갱신까지의 전체 프로세스를 제어합니다.
