# 🧠 AI 음악 추천 스크립트 (`script.js`)

본 프로젝트의 핵심 로직을 담당하는 자바스크립트 파일입니다. 사용자의 입력을 받아 Google Gemini API와 통신하고, 그 결과를 웹 화면에 실시간으로 반영(렌더링)하는 기능을 수행합니다.

---

## ⚙️ 주요 기능 및 로직 흐름 (Logic Flow)

1. **DOM 요소 및 데이터 검증**
   * 입력창(`userInput`), 결과창(`resultBox`), 로딩 스피너(`loader`) 요소를 가져옵니다.
   * 사용자가 아무것도 입력하지 않고 버튼을 누르면 경고창(`alert`)을 띄워 흐름을 차단합니다.

2. **화면 상태 제어 (UI State)**
   * AI가 답변을 생성하는 동안 대기 시간을 시각적으로 보여주기 위해 로딩 스피너를 활성화(`display: block`)하고, 이전 결과창은 숨깁니다.

3. **Google Gemini API 통신 (비동기 Fetch)**
   * 최신 `gemini-2.5-flash` 모델 엔드포인트로 `POST` 요청을 보냅니다.
   * 프롬프트 엔지니어링을 통해 AI가 일반 텍스트가 아닌, 미리 정의된 **유튜브 검색 링크가 포함된 HTML 구조**로 답변하도록 강제합니다.

4. **데이터 정제 및 화면 렌더링 (Data Parsing)**
   * AI 응답에 포함될 수 있는 마크다운 기호(```html ... 
```)를 정규식(`replace`)으로 깨끗하게 제거합니다.
   * 정제된 HTML 소스코드를 `innerHTML`을 통해 결과창 박스에 그대로 주입하여 동적 화면을 완성합니다.

---

## 🛠️ 함수 및 변수 설명 (API Reference)

* **`GEMINI_API_KEY`** (Constant)
  * Google AI Studio에서 발급받은 인증 키 문자열입니다.
* **`getMusicRecommendation()`** (Async Function)
  * 버튼 클릭 이벤트와 연결된 메인 비동기 함수입니다. API 호출부터 예외 처리, 화면 갱신까지의 전체 프로세스를 제어합니다.

# 🎧 AI 음악 추천기 (유튜브 연동 및 공유 기능 업그레이드)

구글 Gemini API를 활용하여 사용자가 입력한 기분과 분위기에 맞춰 음악을 추천해 주는 웹 프론트엔드 어플리케이션입니다. 기존 버전에서 기능 확장 및 안정성을 개선하기 위해 코드를 전면 수정하였습니다.

---

## 🛠️ 이번 업데이트 수정 내용 (Release Notes)

### 1. 음악 개별 공유 기능 추가 (핵심 업데이트)
* **스마트 공유 시스템 도입:** AI가 추천해 준 음악 카드 내에 `🔗 친구에게 공유` 버튼이 동적으로 생성됩니다.
* **디바이스 맞춤형 공유 환경 지원:**
  * **모바일 환경 (`Web Share API`):** 버튼 클릭 시 스마트폰 자체 공유 창이 활성화되어 카카오톡, 문자, SNS 등으로 링크를 즉시 보낼 수 있습니다.
  * **PC 환경 (`Clipboard API`):** 공유창이 없는 PC 브라우저에서는 클립보드에 해당 음악의 유튜브 검색 링크가 자동으로 복사되며, 안내 메시지(Alert)가 출력됩니다.

### 2. 동적 엘리먼트 이벤트 바인딩 오류 수정
* **이슈 사항:** 기존 방식으로는 AI가 통신 이후에 실시간으로 만들어내는(동적 생성) HTML 버튼들에 클릭 이벤트가 작동하지 않는 문제가 있었습니다.
* **해결 방법:** 결과창의 부모 요소(`resultBox`)에 이벤트를 걸어두고 자식 요소의 클릭을 감지하는 **이벤트 위임(Event Delegation)** 기법을 적용하여, AI가 답변을 마친 후 새로 생긴 버튼들도 에러 없이 완벽하게 작동하도록 수정했습니다.

### 3. 기존 HTML 구조 및 로딩 애니메이션 유지
* 기존에 작업되어 있던 초록색 테마의 **원형 로딩 애니메이션(`.loader`)**과 레이아웃 구조를 그대로 보존하였습니다.
* AI가 추천 카드를 생성하는 도중에는 원형 스피너가 회전하며, 추천이 완료되면 카드가 깔끔하게 정렬되어 나타나도록 CSS 및 상태 제어 로직을 다듬었습니다.

### 4. Gemini AI 프롬프트(명령어) 고도화
* AI가 답변을 보낼 때 단순 텍스트나 유튜브 링크만 보내는 것이 아니라, 공유 기능에 필요한 데이터 속성(`data-url`, `data-title`)이 포함된 HTML 태그 구조를 정확히 지켜서 출력하도록 프롬프트 규칙을 강화했습니다.

## 📁 파일 구성 및 적용 방법

* **index.html**: 메인 레이아웃, 기존 원형 로딩 바 및 음악 카드 스타일(CSS) 포함.
* **script.js**: Gemini API 통신 로직 및 동적 공유하기(이벤트 위임) 자바스크립트 코드 포함.

> ⚠️ **실행 전 주의사항**
> * `fixed ai` 파일 최상단의 `const GEMINI_API_KEY = "코드";` 부분에 본인의 실제 구글 API 키를 입력해야 정상 작동합니다.
> * 모바일 기기 본체 공유 기능(`navigator.share`)은 브라우저 보안 정책상 **HTTPS** 환경 또는 **localhost**에서만 팝업이 활성화됩니다. 일반 HTTP 도메인에서는 자동으로 PC 모드(링크 복사)로 전환됩니다.
