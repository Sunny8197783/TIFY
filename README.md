## 🚀 this is for you (AI music playlist service)

> **"그날의 기분을 채팅으로 입력하면, 분위기에 꼭 맞는 맞춤형 노래 플레이리스트를 만들
어주는 AI 음악 추천 서비스."**

---

## 🎯 1. 기획 배경 및 해결 과제
* **현재 상황 및 문제점:**우리는매일다른감정과분위기를경험하지만, 그때그때어울리는음악을직접찾는일은 번거롭다. 기존 스트리밍 서비스의 추천은 청취 이력 기반이라, "지금 비 오는 날의 차분한 기분"처럼 순간의 감 정·맥락을 반영하지 못한다. 결국 사용자는 누군가에게 "이런 분위기 노래 좀 골라줘"라고 부탁하고 싶어진다.
* **우리가 제공할 해결책:**사용자가 채팅창에 오늘의 기분·상황·날씨를 자연어로 입력하면, AI가 감정을 분석해 **맞춤형플레이리스트**를즉시큐레이션한다. 추가로상황태그, AI 위로멘트, 날씨연동, 공유기능을통해단순추 천을 넘어 **"오늘의 감정 동반자"** 경험을 제공한다.
