## ⭐ 3. 핵심 기능
> 프론트엔드 환경에서 3주 내에 구현할 수 있는 핵심적인 기능 딱 3가지를 정의합니다.
1. **[채팅 기반 감정 임력]**: 사용자는 메신저처럼 생긴 입력창에 오늘의 기분을 자유롭게 적는다. 말풍선 UI로 친근하
  게 대화하는 경험을 제공한다.
2. **[AI 감정 분석]**: 입력된 텍스트에서 감정 키워드(기쁨/우울/설렘/차분 등)를 추출해 분위기 카테고리로 매핑한다.
3. **[플레이리스트 추천]**: 분석 결과를 바탕으로 5~10곡의 맞춤 플레이리스트를 카드 형태로 보여주고, 마음에 드는 곡은내리스트에담을수있다.
# 추천 추가 기능 #
