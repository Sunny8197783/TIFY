# TIFY

> AI 감정 기반 음악 추천 웹 서비스 (this is for you)

## 주요 기능
- 💬 채팅 기반 감정 입력 + 12종 감정 칩
- 🤖 AI 감정 분석 (감정별 다른 분석 수치)
- 🎵 감정/날씨 기반 플레이리스트 추천
- 🌤 오늘의 날씨 위젯
- 🔖 곡 저장 (localStorage)
- 💗 감정 달력 + 월별 통계

## API 연결 자리
- `js/weather.js` — OpenWeather API
- `js/ai.js` — OpenAI GPT API
- `js/playlist.js` — YouTube/Spotify Embed

## 실행
`index.html` 더블클릭 또는 `python -m http.server`
